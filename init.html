⚙️ Firestore 資料初始化工具
⚠️ 此頁面僅供初始化使用，完成後請從 GitHub 刪除此檔案。

按下按鈕後，系統將自動建立以下資料：

  • 1 users（建立 collection，含一筆範例結構說明文件）
  • 2 rounds（第 0~5 輪封盤設定，預設 open）
  • 3 teams（48 支隊伍完整資料）
  • 4 results（第 0~5 輪空白結果文件）
▶ 等待執行...
📋 初始化完成，接下來請至 Firebase Console 新增管理員
  1. 前往 Firebase Console → Authentication → 使用者
  2. 點「新增使用者」，填入管理員 email 及密碼，取得 UID
  3. 前往 Firestore → users collection
  4. 找到 _schema 文件，參考欄位結構
  5. 新增一筆文件，Document ID 填入剛才取得的 UID，依照以下欄位填入：
uid : "（從 Authentication 複製的 UID）"
email : "admin@xxx.com"
nickname : "admin"
isAdmin : true
userType : "player"
isApproved : true
createdAt : （點 Timestamp，選當下時間）

完成後即可至 index.html 用管理員帳號登入。
確認系統正常運作後，請刪除此 init.html 檔案。
